# QA Planning Skill

Provides templates for QA documentation.

## Templates

### Test Cases Template
Located at: `.claude/skills/qa-planning/templates/test-cases-template.md`

Use this template for creating:
- Detailed test cases from acceptance criteria
- Priority and type classification
- Step-by-step execution instructions
- Expected results documentation

### Regression Checklist Template
Located at: `.claude/skills/qa-planning/templates/regression-checklist-template.md`

Use this template for:
- Identifying affected areas
- Mapping regression test coverage
- Tracking test execution status

## Usage

When using the QA Agent, reference these templates for consistent output format.

The templates ensure:
- Test cases derived from acceptance criteria
- Complete coverage matrix
- Regression impact analysis
- Clear Definition of Done

## Integration

Used by:
- QA Engineer Agent
- Director Agent (for validation)
- Developer Agents (for understanding test requirements)
