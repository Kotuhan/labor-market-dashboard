# PO Analysis Skill

Provides templates and structured formats for Product Owner analysis.

## Templates

### Problem Statement Template
Located at: `.claude/skills/po-analysis/templates/problem-template.md`

Use this template to structure problem definitions with:
- User problem description
- Urgency and importance
- Impact of inaction

### Acceptance Criteria Template
Located at: `.claude/skills/po-analysis/templates/acceptance-criteria-template.md`

Use this template for Given/When/Then acceptance criteria with:
- Clear preconditions
- Specific actions
- Observable outcomes

## Usage

When using the PO Agent, reference these templates for consistent output format.

The templates ensure:
- Clear problem articulation
- Measurable success criteria
- Testable acceptance criteria in Given/When/Then format
- Explicit scope boundaries

## Integration

Used by:
- Product Owner Agent
- Director Agent (for validation)
